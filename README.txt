¤Comment installer le code?
	¤Pour faire fonctionner le frontend :
	-Ouvrir un terminal sur le dossier principal et exécuter "npm install"
	-Se rendre dans le dossier "frontend" avec le terminal
	-Exécutez "npm run start" pour démarrer le serveur.
	-Le serveur du frontend est à l'URL suivante : http://localhost:4200

	¤Pour faire fonctionner le backend :
	-Ouvrir un terminal sur le dossier "backend".
	-Charger le package nodemon dans le terminal avec : "npm install -g nodemon".
	-Exécutez "nodemon server" pour démarrer le serveur.
	-Le serveur du backend est à l'URL suivante : http://localhost:3000

¤Comment faire fonctionner le code si les packages sont déjà installés?
-"npm start" dans le terminal sur le dossier "frontend".
-"nodemon server" dans le terminal sur le dossier "backend".
-Se connecter à l'URL suivante : http://localhost:4200

¤Comment s'inscrire sur le site?
-Il faut que l'utilisateur ait un mot de passe contenant au minimum : 4 majuscules, 4 minuscules, 2 chiffres.
-Il faut aussi une adresse email.
